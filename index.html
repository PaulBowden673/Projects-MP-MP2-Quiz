Health & Safety Quiz Rules

Answer as many of the questions correctly as you can to earn the highest score. The game will end once all of the questions have been answered.

How To Play:

  • Press Play to start the quiz
  • Once the game starts, you will be given a question, select one of the 4 options to answer the question
  • If you answer correctly, the selected answer will turn green, if answered incorrectly this will turn red
  • For every correct answer, the score will increase by 10 points, incorrect answers get 0 points
  • Once the question is answered, you will be presented with the next question.
  • Your progress is recorded in the top left
  • You can end the game anytime by selecting Quit Game
  • Once all the questions have been answered, the quiz game will end
  • You can save the total score earned during the game by entering a username and clicking Save Score
  • Click Play Again to start the quiz game again
  • Click Return Home to go back to the home screen
  • If you wish to view the highscores table, then from the Home section, select Highscores, only the top 10 scores will be displayed
  • To clear the highscores table of previously recorded scores, click Clear Scores
